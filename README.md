# Tower of Ruin

A PS1-styled dark fantasy Souls-like roguelike developed in Unity.

## 🎮 Game Description

**Tower of Ruin** challenges players to explore a mysterious, ever-shifting tower constructed from the ruins of forgotten worlds. 

### Core Game Pillars:
* **Souls-Like Combat:** Tactical, high-stakes combat featuring stamina management, dodge rolls, lock-on targeting, and precise attacks.
* **Roguelike Progression:** Replayable runs with randomized floor generation, progression loops, and rogue upgrades.
* **PS1 Retro Aesthetic:** Atmospheric, low-poly visuals inspired by classic 3D games of the late 90s.
* **Cooperative Play:** Built with multi-client networking to support future 2–4 player co-op.

## ⚙️ How to Clone & Run the Project

Since this project uses **Git LFS (Large File Storage)** to manage large Unity assets (textures, models, audio), follow these setup steps:

### 1. Install Prerequisites
Make sure both **Git** and **Git LFS** are installed on your machine before cloning:
```bash
# Verify Git LFS is installed and active
git lfs install
```

### 2. Clone the Repository
Clone the repository using HTTPS or SSH:
```bash
git clone https://github.com/Hamachi-300/Tower-of-Ruin.git
```

### 3. Open in Unity
1. Open **Unity Hub**.
2. Click **Add project from disk**.
3. Select the cloned `Tower of Ruin` directory.
4. Launch the project. *(Note: First load will take a few minutes as Unity regenerates the local cache/library folders).*


#### Unity 2021.3 LTS (2021.3.28f1)
