# Tower of Ruin - Agile Roadmap

## Milestone A: Core Gameplay & Mechanics MVP

- [x] **Sprint A1: Movement & Camera**
  - Camera-relative WASD movement.
  - Third-person camera follow with collision detection and close-up transparency fading.

- [x] **Sprint A2: Stamina, Dodge Roll, and Lock-On**
  - [x] Stamina system & UI slider.
  - [x] Dodge Roll movement & stamina cost.
  - [x] Unity 2022.3.62f3 LTS upgrade.
  - [x] URP Package installation & configuration.
  - [x] Lock-On targeting & strafe movement.
  - [x] Dodge Roll invincibility frames (i-frames).

- [ ] **Sprint A3: Basic Attacks & Hitboxes**
  - Light and heavy attack inputs.
  - Attack triggers and collider-based hitboxes.
  - Attack animations and visual feedback.

- [ ] **Sprint A4: Enemies & Health System**
  - Player and enemy health stats.
  - Basic enemy AI (finding player, moving, attacking).
  - Damage logic and invincibility frame validation.

- [ ] **Sprint A5: PS1 Retro Visuals (URP)**
  - Custom Shader Graph for vertex jitter/snapping and affine texture mapping.
  - URP Volume Profile for pixelation downscaling, CRT scanlines, and dithering.

- [ ] **Sprint A6: Multi-Platform Porting & Cross-Platform Testing**
  - Mobile touch controls (on-screen virtual joystick and action buttons).
  - Dynamic UI Canvas scaling for all device aspect ratios.
  - Mobile URP quality asset profile optimization for Android & iOS.
